# mobile-web

这是基于桌面版功能做的手机浏览器批量加水印页面，纯前端静态部署即可。

功能：
- 多图选择
- 右下角文字水印
- 透明 `logo` 叠加
- 文字透明度、字号、偏移调节
- 水印模板预设
- 批量命名规则
- 第一张图实时预览
- 当前图下载
- 批量导出 `zip`
- iPhone 兼容降级导出

部署：
1. 把 `mobile-web` 目录作为 Cloudflare Pages 的输出目录。
2. 不需要 Node、后端或数据库。
3. 首页文件就是 `index.html`。
4. 更正式的部署说明见 `DEPLOY.md`。

说明：
- 图片处理都在浏览器本地完成，不上传服务器。
- 批量打包使用前端原生 `zip` 生成逻辑，适合静态站点。
